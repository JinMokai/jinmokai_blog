# 程运熙

在<TextLink text={"鄱阳中学历任校长"} link={"/blogs/鄱阳中学历任校长"}/>有简单介绍。

程运熙，生于1862年，卒年不详

光绪二十三年（1897 年考取举人）（江西科举第九十六名）

在1904-1905年担任江西省饶州府中学校学校校长（现鄱阳中学）
